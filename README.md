## Projects

### Refinery Synthetic Data

Synthetic data generator for refinery operations. Produces realistic time-series (PI tags), lab samples (LIMS), and ground-truth labels for AI/ML development.

- **Phase 1 — CDU Prototype** (`AISolutions/Refinery Synthetic Data/Phase-1-Cdu-Prototype/`)
  Mock-physics CDU generating PI + LIMS data with a fouling fault scenario. Shipped.
- **Phase 2** — multi-source integration (planned)
- **Phase 3** — DWSIM and IDAES integration (planned)
- **Phase 4** — production platform (planned)

See the README inside each phase folder for details.
