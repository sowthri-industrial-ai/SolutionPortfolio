# Refinery Twin — NVIDIA Omniverse + Isaac Sim

A real Physical AI demo: Crude Distillation Unit twin in Omniverse Kit
with live process data binding, plus Isaac Sim scenarios for safety
and inspection use cases.

## Status

In-build. Currently in Phase 0 (foundation). See
[`docs/charter/PROJECT_CHARTER.md`](docs/charter/PROJECT_CHARTER.md)
for scope and [`docs/OPERATOR_RUNBOOK.md`](docs/OPERATOR_RUNBOOK.md)
for the live operations log.

## What's here

- `asset-library/` — parametric USD builders for refinery equipment
- `data-fabric/` — process simulator, in-memory event bus, SSE endpoint
- `kit-extension/` — the Omniverse Kit extension (`com.sowthri.cdutwin`)
- `isaac-scenarios/` — gas dispersion, inspection rover, valve dynamics
- `docs/` — project charter, operator runbook, architecture notes
- `docs/charter/` — binding planning artifacts (charter, roadmap, agreements)
